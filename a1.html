Schematic and Calculations

I decided to try adding a blue LED as a challenge. I calculate the resistance for the green and yellow LEDs as 160, and 85 for the blue. This makes sense, since blue requires a higher voltage and therefore a lower resistor. To be safe, I rounded each number up for the resistor I ended up choosing: 220 for yellow/green, and 100 for blue. I also experimented with a higher resistor to make the blue bulb a bit dimmer. I recreated the circuit with a 1k resistor and you can see in the gif that the blue is slightly dim.

Firmware


        // the setup function to initalize all pins used in the circuit as outputs
        void setup() {
          // initialize pin 8 as an output
          pinMode(8, OUTPUT);
          // initialize pin 7 as an output
          pinMode(7, OUTPUT);
          // initialize pin 4 as an output
          pinMode(4, OUTPUT);
        }

        //Sets up a loop that runs indefinitely
        void loop() {
          //Turn yellow LED (pin 7) on
          digitalWrite(7, HIGH);  
          //One second pause
          delay(1000);  
          //Turn yellow LED off                   
          digitalWrite(7, LOW);   
          //One second pause
          delay(1000);  
          //Turn green LED (pin 8) on
          digitalWrite(8, HIGH);  
          //One second pause
          delay(1000);   
          //Turn green LED (pin 8) off                
          digitalWrite(8, LOW);   
          //One second pause
          delay(1000); 
          //Turn blue LED (pin 4) on
          digitalWrite(4, HIGH);  
          //One second pause
          delay(1000);   
          //Turn blue LED (pin 4) off                   
          digitalWrite(4, LOW);   
          //One second pause
          delay(1000);        
        }
